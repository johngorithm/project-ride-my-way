Welcome

This is RIDE MY WAY API. This version one of the api covers the following end-points



GET: /api/v1/rides

get all available ride offers


GET: /api/v1/rides/:rideId

get information about a specific ride offers


POST: /api/v1/users/rides

creates a new ride offer


POST: /api/v1/rides/:rideId/requests

join a ride offer


GET: /api/v1/users/rides/:rideId/requests

Get all requests made to a ride offer


PUT: /api/v1/rides/:rideId/requests/:requestId

Accepts or Rejects a ride request.
This endpoint requires an `action` query - `accept` or `reject`


POST: /api/v1/auth/signup

Register a user


POST: /api/v1/auth/login

Login a user
