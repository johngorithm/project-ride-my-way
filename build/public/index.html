Welcome

This is RIDE MY WAY API. This version one of the api covers the following end-points



GET: /api/v1/rides

returns all available ride offers


GET: /api/v1/rides/:ride_id

returns a ride offer with :ride_id


POST: /api/v1/rides

creates a new ride offer


POST: /api/v1/rides/:ride_id/request

sends a request to join a ride offer
